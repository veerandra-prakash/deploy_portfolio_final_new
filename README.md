# Veerandra Prakash — Portfolio

Production-ready portfolio built with React + Vite + Tailwind CSS + Framer Motion.

## Requirements

- **Node.js 16+** (works on 16, 18, 20, 22)
- npm 7+

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3 (PostCSS)
- Framer Motion
- React Icons
- Lenis (smooth scroll)
